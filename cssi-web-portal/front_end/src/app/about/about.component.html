The CSSI_Web Portal is a link between scientist and data stored on the NvWex cyber infastructure with a stated goal of gathering data to better handle wildfires and other natural disasters. The NvWex cyber infastructure is a structure that allows organizations to set up LoRa devices in the Greater Nevada area that can send sensor data from teh environment back to secured servers of the NvWex system. The CSSI-Web Prtal provides access to the sensor data. Creating tables, graphs, and maps. Provides statistical data, and exports infromation in csv and png files for external data processing.
