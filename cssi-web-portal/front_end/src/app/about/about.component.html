The CSSI Web Portal is desiged to be a data portal for researchers, utilizing LoRa devices set up across the Greater Nevada Area to analyze climate data, Initally created for researcher, the portal is planned to extend to the general populace, utilizing sensors for comuunity-wide alerts on threats like wildfires, enhancing local saftey. The project utiliizes an Angular-based framework, HTML, CSS, and TypeScript to create its front-end design. The backend structure consists of Apache acting as a middleware, Python for the backend logic, PostgreSQL for database management, with a Linux operating system, and a planned nginx load balancing system.
